Create a view to display available titles

Once the Title object had a status field, Adam could create a view to filter records by status and show only available titles. Then, he modified the selector page to use this view so that librarians can only add available titles to a new Check Out record.

Follow these steps to provide this functionality:

  1. Create a view that filters titles based on their status:
    1. Navigate to the application Titles page.

    2. From the Viewing: menu, select Create New View.

    3. For View Name, enter Available Titles.
    4. Make sure that the Show In field has all devices checked.
    5. The Selected Columns box should already contain Title. Add the Author(s), Publisher, Publication Date, and Status fields by moving them from the Available Columns to the Selected Columns list. Arrange them in a logical order. Note that in a real application, you might not include the Status field in this view. We are including it here so you can verify that the filter works.
    6. Scroll to the View Filters section and set the following:
      • From the drop-down menu in row 1, select Status
      • From the second menu, select equals
      • In the empty field, enter Available.
    7. For Permissions, make sure Administrator, Library Employee, and Library Member are selected.
    8. Click Save.
      The Titles page displays the new view, which should only display records with a status of Available.
  2. Set the new view to be the default for the title selector.
    1. Open the Titles object definition.
    2. From the shortcut bar, select Pages.
    3. In the Selector row, click Edit.

    4. Roll your cursor over the List of Titles component and click the arrow to display the Properties menu.

    5. For the Use List View field, select Available Titles.

    6. Deselect all other options, so the menu looks something like this:

    7. Save your changes.
  3. Make the All Titles view the default view for the Titles page:

    1. Return to the app Titles page and open the Page Options menu:

    2. Select Design This Page and open the Properties menu for the List of Titles component.
    3. For the Use List View field, select the All Titles view and leave the other options with their default values.
    4. Save your changes.
  4. Test by creating a new check out record. Notice that when you click the search icon to pick a title, only available titles display.
