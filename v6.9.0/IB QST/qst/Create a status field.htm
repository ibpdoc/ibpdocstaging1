Create a status field

Platform provides a sophisticated workflow framework, that allows you to automate processing as a record moves through different states. Titles will only have one of two states, checked out or available. So, Adam decided not to use workflows. Instead, he created a new field on the Title object to store the status. Then he created a trigger on the Check Out object to change the value of this field when a title is checked out.

To reproduce his steps:

  1. Add a status field to the title object:
    1. From the application menu, select Titles.
    2. From the page menu, open the Title object definition.
    3. In the Fields section, click New Field.
    4. Select the Text type and click Next.
    5. For Field Label, enter Status
    6. For Default Value, enter Available.
    7. Leave the other fields with their default values and click Save.
      The Add to Pages and Views page opens.
    8. De-select all of the Application Pages except for View Title and Edit Title
    9. For Views, select All Titles.
    10. Click Save.
  2. Test the new attribute:
    1. Return to the app and create a new Title record.
      The new record should look similar to the following:

    2. To make the other record(s) consistent, enter Available in the Status field for titles that are not currently checked out. (Edit only the Status field, leave the Check Out field empty.) You can edit records from the Actions menu or by using the inline editor:

      • From the Actions menu select Edit to change the value:

      • To edit inline, double-click the cell, enter the value, and click the check mark to save the value:
