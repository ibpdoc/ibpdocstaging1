Change the menu icons

Platform provides menu icons from two font libraries, or you can upload your own. Adam used symbols from the built-in libraries. Follow these steps to change the Platform menu icons for your app:

  1. From the Application Switcher, select App Settings.

    The Setup Application page opens.
  2. Change the icon for the Check Outs page (tab):
    1. Navigate to the Tabs section.
    2. Click Edit in the Check Outs row.
      The Tab: Check Outs screen opens.
    3. For the Icon Type field, select Font icon.
    4. For the Font classes field, select Bootstrap Glyph.
    5. Click the dropdown menu (on the right side of the field) to view the available symbols.
    6. Enter check in the Search icon field and select the box with a check mark.

    7. Save the change and return to the application to view the results.

  3. Repeat the steps and pick symbols for the other tabs.

    Use any symbol you like. Adam chose to use these symbols (some are from the Font Awesome library).
