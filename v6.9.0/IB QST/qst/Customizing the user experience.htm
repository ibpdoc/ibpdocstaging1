Customizing the user experience

Platform provides many ways to customize the user interface and improve the user experience. You can choose from a palette of templates; add pages, add your own logo, header, and footer; and you can modify application pages using JavaScript and HTML. You can also customize the experience based on the user's device.

Making sure that the app supports user goals will help encourage end users to adopt it. For example:

  • When users log into the app, what will they see?
  • Will they know how to get started?
  • Can non-critical elements be removed to de-clutter screens and avoid confusion?
  • Where can they find help?

Although Platform provides a navigation framework, you can improve the user experience by:

  • Adding a welcome page to provide orientation.
  • Changing the order of tabs on the application menu bar and associating icons with them.
  • Adding, removing, or moving fields in views or on pages such as the new record page.
  • Adding field-level help text.
