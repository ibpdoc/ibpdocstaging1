Add fields to list views

When you create objects manually or using the Quick Create wizard, the default list view of all records only displays the following fields: record name (For example, Title or Check Out), Updated At, and Updated By.

You can easily configure the All Titles and All Check Outs views to include the fields of interest to end users. Views are part of the object definition.

Follow these steps to modify the view of all titles:

  1. Open the Title object definition:
    1. From the Platform menu, select Titles.

    2. Open the page menu and select Object Definition.

    The object definition page opens:

  2. Edit the All Titles view:

    1. From the shortcut bar, click the link to jump to the Views section.

    2. In the View Name column of the Views section, click All Titles.
    3. Click Edit.
      The Edit View page opens. Note, your page might also have a Delete button.

    4. From the Available Columns list, select Author(s) and use the arrow to move it to the Selected Columns list.
    5. Do the same for Publisher and Publication Date.
    6. In the Selected Columns list, use the arrow controls on the right to put the fields in a logical order.
      For example, Adam arranged them like this:

    7. Click Save.

    8. Click return to Lending Library. Find this shortcut link under the search box in the top right corner:

    9. Check the Titles page to see the modified view.
  3. Follow the same procedures to add fields to the All Check Outs view, but use a different navigation path:
    1. From the application menu, select Check Outs.

    2. Expand the Viewing: All Check Outs menu:

    3. Next to the All Check Outs menu item, select Edit this View:

    4. Add the Date Borrowed, Date Due, Date Returned, and Titles, fields to the view, order them logically, and save the view.
