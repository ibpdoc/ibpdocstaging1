Managing Infinite Blue Platform License

This section provides detailed information on requesting license support or submitting a new/upgrade license order form.

Requesting license support

You may seek assistance for any of the following scenarios:

  • Upgrading an existing 5.x environment to 6.x

  • Experiencing an error or a problem with one of your current licenses

  • Updating the details of the license administrator who receives license key emails

You can request license support by filling up the form available here.

Submitting new/upgrading an existing license

You can fill out the license request form for any of the following scenarios:

  • Requesting a new license.

  • Modifying an existing license.

You can submit a new/upgrade an existing license by filling up the form available here.

Email requests for licence upgrades are no longer accepted. Make sure that the relevant request forms for licence requests are filled out appropriately.
Note:
  • From June 30, 2023, Infinite Blue Platform will no longer be issuing 5.x licenses and issue renewals for 6.x licenses only. Existing 5.x customers are advised to upgrade the Platform version to 6.x as their licenses are set to expire on January 31, 2024.
  • For detailed information on platform upgrades, see Upgrading Private Cloud to Version 5.6 until 6.1 or Upgrading Private Cloud to Version 6.2 and above.
