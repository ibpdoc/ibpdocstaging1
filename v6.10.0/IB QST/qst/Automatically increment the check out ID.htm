Automatically increment the check out ID

When you create an object, Platform adds a field with the object name. This field is of type Record Name. For example, the Title object has a Title field and the Check Out object has a Check Out field. While it makes sense to put the title in this field, it would be less intuitive for a librarian to have to assign a name to each new Check Out record. Instead, Adam would prefer to have the value populated automatically.

Platform has an Auto-Number field type. Adam can use the value Platform assigns to this field as the ID (Record Name) for Check Out records. The Record Name Template inserts values in this field based on a formula. So, Adam first added an Auto-Number field to generate the values and then used the Record Name Template to insert them into the Record Name field.

Re-create Adam's steps as follows:

  1. Open the Check Outs object definition:
    1. From the Platform menu, select Check Outs.

    2. From the page menu, select Object Definition.

    The Check Out object definition opens.
  2. Click Fields in the shortcut bar or scroll to the Fields section.

  3. Click New Field.

    The New Field screen opens:

  4. Scroll to the Advanced Type list, select Auto-Number, and click Next.
    The properties screen opens.

  5. Define the properties as follows, leaving the other fields with their default values:
    1. For the Field Label, enter Autonum.
    2. For Display Format, enter {0000000}.
    3. For Starting Number, enter 0000001.
    4. Click Assign values for all existing object records now. This will populate the fields for any existing Check Out records.
    5. Click Save.
      The Add to Pages and Views screen opens.
    6. Since users will not edit the values in this field and they will be inserted into another field, there is no need to include them on any page or view. Clear the check marks from all boxes.
    7. Click Save.
    The Check Out object definition opens again.
  6. Use a formula to insert the value of the new field in the Record Name Template:
    1. On the Check Out object definition page, click Edit.
      The screen for editing object properties opens. The Template Helper can generate a token that obtains the value from the Autonum field.

    2. From the Select Merge Token menu, select Autonum.
      {!Autonum} displays in the second field. Platform replaces this token with the values from the Autonum field.
    3. Copy the token and paste it in the Record Name Template field.
    4. Select Update names of all existing object records using this template. (Depending on the state of your application, this box might be labeled: Assign values for all existing object records using this template)
    5. Click Save.
  7. Test the implementation by creating a new Check Out record:
    1. Click return to Lending Library. Find this shortcut link under the search box in the top right corner:

      The Check Outs page opens.
    2. Click New Check Out.
      The New Check Out page opens.
    3. For Date Borrowed, use the calendar and pick a date.
    4. Leave the Date Due and Date Returned fields blank.
    5. For Title, click the search icon and select a title that you created previously, click Attach Selected.
    6. Save the new record.
      The Check Out field should have a 7-digit numeric value.
