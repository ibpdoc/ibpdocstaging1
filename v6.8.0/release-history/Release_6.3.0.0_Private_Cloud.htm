Known Issues

Following is the list of known issues with this release:

  1. [REST API 2.0] – Ability to update storage via REST API 2.0 PUT call, even if the storage value is a read only field. Considering that, this field must be ignored during the update. If not, the user may lose the data stored in the storage.

    Workaround: The user needs to re-execute the PUT call with the original storage location.

  2. [Charts][PostGre] – When a chart is created with GroupBy as Day then the dates are shown improperly on the chart.
