Test the check out process

Test your app by creating a new check out record:
  1. Return to the app Check Outs page.
  2. Click New Check Out.
    The New Check Out page opens.

  3. Leave the Date Borrowed, Date Due, and Date Returned fields empty. Later, we'll remove them from the New Check Out page since the borrowed and due dates are set automatically and it doesn't make sense to set the returned date in a new record.
  4. In the Titles field, click the lookup icon.
    The Titles Selector displays.
  5. Select a record (if multiple records are available, you will need to click Attach Selected).
    The title displays in the Titles field.
  6. Do the same to select a User.
  7. Save the record.
    The new record should look something like the following, and include the automatic ID, date borrowed, and calculated due date.

If you create multiple Check Out records, you will notice that all title records are always available in the selector window, even those that are already checked out. See how Adam addresses this issue by Changing title status and displaying available titles.
