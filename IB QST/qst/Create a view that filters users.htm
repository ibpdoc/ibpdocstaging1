Create a view that filters users

Since Adam does not want employees to be able to create administrator or employee records, he created a view that filters User records. Then he set this view to appear on the Library Members page, the New Record page, and the User selector page.

To ensure that employees can only see member records, log in with your account that has administrative privileges and follow these steps:

  1. Create a new view of User records:
    1. From the Platform app, navigate to the Views section of the User object definition.
    2. Create a new view.
    3. In the View Name field, enter Library Members.
    4. In the Columns section, remove User Name and add the following fields: First Name, Last Name, User Name, and User Role [role].
    5. Scroll to the View Filters section and set a filter on the User Role [role] field to include only users with a role of Library Member as shown in the following screen:
    6. Scroll to the Permissions section and verify that the Library Employee role has View permission.
    7. Save the view.
  2. Make this new view the default view for the Library Members tab in your application:
    1. Return to the Lending Library application.
    2. Open the Library Members page.

    3. From the Page Options menu, select Design This Page.

    4. Roll your cursor over the List of Users component and click the arrow to display the Properties menu.
    5. For Use List View, select the Library Members view.
    6. For the rest of the options, leave only the Show New Link option selected.
      The properties should look like the following :

    7. Save the page.
  3. Set the Library Members view to display when employees create a new Check Out record:
    1. Open the Check Outs page and click New Check Out.
      The New Check Out page opens.
    2. From the Page Options menu, select Design This Page.
    3. Roll your cursor over the User field and open the Properties menu.
    4. For Use List View, select Library Members and leave all boxes unselected.
      Click the thumbnail to see an example.

    5. Save the page.
    6. Click the search button next to the User field and verify that only library members display.
  4. Log in as a user with the Library Employee role and verify that only library members display when creating a new check out.
