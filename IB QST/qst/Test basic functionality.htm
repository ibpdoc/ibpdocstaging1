Test basic functionality

Let's take a look at the application so far. In just a few minutes you created the app foundation to store titles, check out information, and user accounts. This is a great start! Take the app for a test drive by creating title and check out records:

  1. If you are not viewing Lending Library, use the application switcher to open it.
  2. Create at least three title records:
    1. From the Platform menu, select T to open the Titles page.

    2. Click New Title.
      The New Title page opens. This is one of the pages that Platform created for you. Later in the tutorial, you will edit this page to fit the application workflow.
    3. Enter values for the Title, Author(s), and Publisher fields and select a Publication Date from the date picker. Leave the Check Outs field empty. This field will be filled in automatically when you add this title to a check out record.
    4. Click Save & New and add at least two more titles. After the last record, click Save (the orange button with the check mark).
      The page displays the default All Titles view that Platform created for you. The fields you created in the Quick Create wizard do not show in this view. You will address that for both titles and check outs later. After you add a few records, the view on the Title page should look similar to this:

  3. Create a Check Out record:
    1. From the Platform menu, select C to open the Check Outs page.
    2. Click New Check Out.
      The New Check Out page opens. Notice that the Titles field is different. It has two controls, one for looking up a related title and one for creating a new title. Later, we will create a relationship with the User object so that this page will have similar controls to associate a library member with the record.

    3. Click the lookup icon for Titles.
    4. Select a record and click Attach Selected.
      The title displays in the Titles field.
    5. Leave the Date Returned field empty.
    6. Click Save (the check mark).
    Note that you could not specify the date of the check out or the due date. You will take care of these limitations when you add business logic.
You can add more Title and Check Out records if you like. Pretty cool for the amount of effort, right? Next, follow the steps to add fields to the view of all titles..
